Noto Serif Display Variable Font
================================

This download contains Noto Serif Display as both variable fonts and static fonts.

Noto Serif Display is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in these files:
  NotoSerifDisplay-VariableFont_wdth,wght.ttf
  NotoSerifDisplay-Italic-VariableFont_wdth,wght.ttf

-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
